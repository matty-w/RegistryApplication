Help: Adding To And Removing Registry Files

Description:

This is the Registry Web Application, from here, Users are able to alter, add or delete registry entries from the various projects available to them as they please.

Project Combobox:

From here, the User is able to select the Project they wish to edit. This is done by selecting the Project desired from the Drop Down list, and then pressing the 'Search' button, the corresponding table will then appear on the screen.

Registry Table:

The Registry Table will only appear once a valid Project has been selected from the Project Combobox, and then searched. If the Project you select doesn't contain any Registry Entries, you will be prompted to select another. Otherwise, a table will be displayed which contains the Catalouge Keys, the associated File Path for these Keys, and also a checkbox in each row, which corresponds to the row it lays in.

Delete Registry Button:

The Delete Button, located just below where the Registry Table lies, will be used to delete the Registry Entries you want to remove from the Project. This is simply done my 'checking' the appropriate checkbox located in the row of the entry you wish to delete in the Registry Table, and then pressing the 'Delete' button. The Application will then prompt you as to whether you are sure you wish to delete the Entries you have selected, if you then select 'Yes' from this prompt, the Application will delete the entries. If you have changed your mind, and no longer wish to delete the entries, simply select 'No' from the prompt.
